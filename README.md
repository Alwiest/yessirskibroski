## Приложение для заметок

## Описание проекта
Веб-приложение для создания и хранения заметок с аутентификацией пользователей. 
Реализовано по техническому заданию с использованием Flask, SQLAlchemy и Bootstrap 5.

## Функциональность
- Регистрация и аутентификация пользователей
- Главная страница со списком всех пользователей и количеством их заметок
- Просмотр заметок каждого пользователя
- CRUD операции для заметок и пользователей (админом)
- Административная панель (суперпользователь)
- Поиск по тегам и ключевым словам

## Технологический стек
### Backend
- **Flask** - веб-фреймворк
- **Flask-SQLAlchemy** - ORM для работы с БД
- **Flask-Migrate** - миграции базы данных
- **SQLite** - база данных 

### Frontend
- **Jinja2** - шаблонизатор
- **Bootstrap 5** - стилизатор
- **HTML5** - разметка страниц

### Архитектура
- **MVC (Model-View-Controller)** - архитектура приложения
- **Git Flow** - методология ветвления

## Установка и запуск
1. Клонировать репозиторий
2. Установить зависимости: `pip install -r requirements.txt`
3. Инициализировать базу данных: `flask db upgrade`
4. Запустить приложение: `python run.py`
5. Открыть в браузере: `http://localhost:5000`

## Модели данных
### Пользователь (User)
- id (Integer, primary_key)
- username (String, уникальный)
- email (String, уникальный)
- password (String)
- is_admin (Boolean) - суперпользователь
- notes (relationship) - связь с заметками

### Заметка (Note)
- id (Integer, primary_key)
- title (String) - заголовок
- content (Text) - текст заметки
- created_at (DateTime) - дата создания
- updated_at (DateTime) - дата последнего изменения
- tags (String) - теги через запятую
- user_id (Integer, ForeignKey) - автор заметки

## Командная разработка
Проект разрабатывается командой из 3 человек с использованием Git Flow:
- `main` - стабильная версия
- `develop` - ветка разработки
- `feature/*` - ветки для новых функций

Каждый участник работает в своей ветке, изменения мержатся в develop через Pull Requests.
